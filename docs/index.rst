Welcome to easy-sfx-manager's documentation!
============================================

**Easy SFX Manager for Godot Engine 4.x.**
Easy SFX Manager (ESF for short), is an audio manager, utilizing and operating positional and non-positional audio in 3D projects. Written in GDscript and planned to be updated with time as C++ GDExtension.
Current iteration is a very basic, robust solution to use as Singleton or as a scene Node based on user scenario.

Check out the :doc:`usage` section for further information, including
how to :ref:`installation` the project.

.. note::

   This project is under active development.
